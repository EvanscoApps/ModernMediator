# ModernMediator Benchmarks

Benchmarks compare ModernMediator against MediatR 12.4.1
(the last MIT-licensed version of MediatR).

Run benchmarks locally:
```bash
cd benchmarks/ModernMediator.Benchmarks
dotnet run -c Release
```

Results are saved to `BenchmarkDotNet.Artifacts/`.
